En utilisant l'image ci-dessus (lien), vous allez réaliser une image réactive.

Question 1

Ecrire une page HTML contenant cette image.

Question 2

Rendre l'image globalement active : en cliquant sur l'image, on arrive à la page wikipedia sur la France.

Question 3

En utilisant la balise <map>, rendre l'image réactive. Vous pouvez-vous inspirer du tutorial suivant : W3School map tag

On définira les zones suivantes :

  • Une zone circulaire englobant Brest renverra vers une page Web contenant le texte suivant : "Bienvenue à l'ISEN Brest"
  • Une zone hexagonale englobant Lille renverra vers une page Web contenant le texte suivant : "Bienvenue à l'ISEN Lille"
  • Une zone circulaire englobant Rennes renverra vers une page Web contenant le texte suivant : "Bienvenue à l'ISEN Rennes"
  • Une zone rectangulaire englobant Toulon renverra vers une page Web contenant le texte suivant : "Bienvenue à l'ISEN Toulon"
  • Une zone circulaire englobant Nantes renverra vers une page Web contenant le texte suivant : "Bienvenue à l'ISEN Nantes"
  • Le reste de l'image renverra vers une page Web contenant le texte suivant : "L'ISEN ouvrira très prochainement dans votre ville "

Les formulaires

Soit un sondage permettant de savoir comment vous avez connu l'école.

Ecrire un formulaire HTML comprenant les champs suivants à renseigner (choisir l'outil le plus approprié pour chacun) :

  • Votre nom et prénom
  • Date de naissance (menus déroulants)
  • Ce qui vous a fait connaître l'école : un ami, un salon, une porte ouverte, autre (pas en menu déroulant)
  • La possibilité de télécharger votre CV à partir d'un fichier
  • Commentaires (texte libre)
  • ...

Une fois ce formulaire renseigné, un bouton permettra d'envoyer vos réponses par mail à l'adresse de l'enseignant (ou la vôtre pour test). Le mail généré devra avoir comme sujet : "Sondage ISEN". Le CV ne pourra pas apparaître comme pièce jointe dans le message mais on doit visualier le nom du fichier.
