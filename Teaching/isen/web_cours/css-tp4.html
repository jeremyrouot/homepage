Introduction aux technologies WEB

TP 4 CSS Positionnement

L'objectif de ce TP est d'utiliser le positionnement pour styler une page web.

Général

Pour ce TP on utilisera les fichiers suivants :

  • position.html
  • chaton.png

Créer un fichier position.css et le lier à la page html.

Question 1 : Barre de navigation

Pour la balise ul de la barre de navigation :

  • Supprimer la puce (style)
  • Mettre le padding à 10px
  • Mettre le margin à 0px
  • Mettre la couleur de fond en noir
  • Augmenter la taille des fontes de 50%

Pour les balises li de la barre de navigation :

  • Mettre le padding à 10px

Pour les balises a de la barre de navigation :

  • Supprimer le style souligné
  • Mettre la couleur du texte en blanc

Lorsque les balises li sont "survolées" par la souris mettre la couleur de fond en gris.

Supprimer ce comportement pour le premier li (Pouce levé).

Changer le positionnement des balises li pour que les éléments se retrouvent sur la même ligne.

Question 2 : Position de la barre de navigation

Styler la balise nav pour que son positionnement soit fixe en haut à gauche de la page avec une largeur de 100%.

Corriger la marge haute de la balise div#content pour que son contenu ne soit pas sous la barre de navigation.

Question 3 : Image chaton

Utiliser le positionnement flottant pour repousser à droite l'image du chaton.

Question 4 : La pub !

Faire les changements suivants sur la balise div ayant la classe pub :

  • Placer l'élement de manière absolu au milieu de la page.
  • Mettre le fond de l'élément en violet
  • Mettre la couleur du texte en blanc
  • Donner à l'élement une hauteur et une largeur de 200px
