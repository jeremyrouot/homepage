TP 4 Javascript

Chronomètre

Préparation

Créer un fichier HTML vierge.

Créer dans ce fichier un élément div> contenant le texte suivant : «Chronomètre : ».

Chronomètre en secondes

Ecrire une fonction javascript indiquant le nombre de secondes écoulées depuis le chargement de la page.

Le résultat de cette fonction apparaîtra à la suite du texte «Chronomètre : ». La valeur doit changer à chaque seconde.

Indication : Vous pouvez utiliser les méthodes de temporisation de l'objet window setTimeout(), setInterval()

Chronomètre en minutes et secondes

Faire évoluer le programme précédent pour que l'affichage soit maintenant en minutes et secondes.
