C'est quoi le WEB ?

Le World Wide Web (WWW), littéralement la « toile (d’araignée) mondiale », communément appelé le Web, et parfois la Toile, est un système hypertexte public fonctionnant sur Internet. Le Web permet de consulter, avec un navigateur, des pages accessibles sur des sites. L’image de la toile d’araignée vient des hyperliens qui lient les pages web entre elles.

Internet

Internet est le réseau informatique mondial accessible au public. C'est un réseau de réseaux, à commutation de paquets, sans centre névralgique, composé de millions de réseaux aussi bien publics que privés, universitaires, commerciaux et gouvernementaux, eux-mêmes regroupés en réseaux autonomes (il y en avait 47 000 en 2014).
L'information est transmise par Internet grâce à un ensemble standardisé de protocoles de transfert de données, qui permet l'élaboration d'applications et de services variés comme le courrier électronique, la messagerie instantanée, le pair-à-pair et le World Wide Web.
L'internet ayant été popularisé par l'apparition du World Wide Web (WWW), les deux sont parfois confondus par le public non averti. Le World Wide Web n'est pourtant que l'une des applications d'internet.

Historique

  • 1959-68 : ARPA naît de la guerre froide (peur de la guerre nucléaire)
    • Faiblesse d'un système centralisé
    • Réseau maillé : ARPANET, NPL
  • 1970-82 : Ouverture sur le monde
    • Apparition du courrier électronique (1972)
    • Apparition de TCP/IP (1974)
  • 1983 : TCP/IP adopté comme standard d'ARPANET
  • 1983 : Expansion du réseau
    • Réseaux hétérogènes (NCP et TCP/IP)
    • Utilisation principale : scientifique
  • Années 1990 : Explosion d'Internet
    • 1989 : Tim Berners Lee (CERN) crée le World Wide Web
    • 1990 : 1ier navigateur Web appelé World Wide Web
    • 1993 : 1ier navigateur Web graphique appelé NCSA Mosaïc
    • 1995 : Ouverture au grand public

Technologies

Pré-existantes

  • TCP/IP
  • DNS
  • MIME
  • Codage des caractères, formats d’image,...

Base

  • URL : identification des ressources
  • HTML : pages web, langage basé sur SGML
  • HTTP: protocole de communication entre les navigateurs et les serveurs Web (indique le type MIME des ressources transférées)

Statique

  • HTML, images, son, vidéo, animations Flash
  • XML, XHTML : obsolètes
  • CSS (feuilles de style en cascade) : langage décrivant la présentation des documents
  • DOM : interface indépendante de tout langage permettant la mise à jour du contenu, de la structure et du style du document

Dynamique

Coté client

  • Javascript : langage de scripts permettant de rendre les pages interactives
  • Applet : applications écrites en Java
  • ActiveX : applications écrites en C++, Visual Basic ou Java
  • Plugin : son, vidéo, applications

Coté serveur

  • CGI : protocole de communication inter-processus entre le serveur HTTP et des applications externes
  • PHP : langage de programmation permettant de générer des pages web dynamiques , forte intégration avec le serveur HTTP et le langage HTML
  • ASP : ensemble de logiciels développés par Microsoft permettant de créer des pages dynamiques (fonctionne sur plate-forme Internet Information Services)
  • Servlets/JSP : technique basée su le langage JAVA permettant de générer des pages web dynamiques (serveur Tomcat)
