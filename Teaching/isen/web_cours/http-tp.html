Coté serveur

Pour simuler un serveur WEB on utilisera la commande : 

nc
(pour netcat) avec la syntaxe suivante : 
nc -l localhost port

Il est ensuite possible de se connecter à ce serveur via un telnet ou un navigateur WEB.
