Timer en JavaScript

Fonction setTimeout

  • Appartient à l'objet window
  • Syntaxe setTimeout(nom_fonction, temps_en_ms);

Fonction setTimeout


var maVariable = setTimeout(monTimer, 2000);

function monTimer() {
	var d = new Date();
	console.log(d.toLocalTimeString());
}

Fonction setTimeout

Annulation avec la fonction clearTimeout

Fonction setTimeout


var maVariable = setTimeout(monTimer, 2000);

function monTimer() {
	var d = new Date();
	console.log(d.toLocalTimeString());
}

function stopTimer() {
	clearTimeout(maVariable);
}

Fonction setInterval

  • Idem mais appel à interval régulier
  • Annulation avec clearInterval

Canvas

  • Apparait avec le HTML 5
  • Permet de définir une zone rectangulaire où il est possible de "dessiner" à l'aide de JavaScript
  • Lancé en 2004 dans Safari et présent depuis 2005 dans Firefox

<canvas id="monCanvas" width="200" height="200">
	Texte alternatif pour les navigateurs ne supportant pas la balise canvas
</canvas>

Pour le tracé

  • Initialisation beginPath()
  • Point de départ moveTo(x,y)
  • Point d'arrivé lineTo(x,y)
  • Clôture de la forme closePath()
  • Affichage du contour stroke() ou remplissage fill()

var ctx = c.getContext("2d");
ctx.beginPath();
ctx.moveTo(10,10);
ctx.lineTo(90,90);
ctx.moveTo(90,10);
ctx.lineTo(10,90);
ctx.closePath();

ctx.stroke();
  • Propriété fillStyle ou strokeStyle : couleur CSS
  • Propriété lineWidth : épaisseur du trait
  • Propriété lineCap : style de fin de ligne

ctx.strokeStyle = "blue";
ctx.lineWidth = 2;

Forme rectangulaire


ctx.fillStyle = "green";
