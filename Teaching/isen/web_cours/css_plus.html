Media queries

On a vu qu'il était possible de spécifier un média de destination pour une ou plusieurs feuilles de style.


<link rel="stylesheet" media="screen" href="pour_ecran.css" type="text/css'">
<link rel="stylesheet" media="print" href="pour_impression.css" type="text/css'">
						

@media print {
	nav, footer, aside { display: none; }
	body { font-size: 120%; color: black; }
}

Avec CSS3 il est également possible de spécifier des expressions sur des critères


<link rel="stylesheet"screen and (max-width: 640px)"screen" href="pour_petit_ecran.css" type="text/css'">
						

@media screen and (max-width: 640px) {
	nav, footer, aside { display: none; }
	body { font-size: 70%; }
}

Les opérateurs :

  • and
  • only
  • not

Les critères (pouvvant être préfixés par min- et max-) :

  • width, height, resolution
  • color, color-index, monochrome
  • aspect-ration
  • ...
