Les évènements

Les actions sont liées à des évènements (utilisateur, chargement de la page, fermeture de la page,...). Le but de l'exercice est de comprendre comment déclencher l'exécution d'une fonction javascript suite à un évènement.

Fonctions javascript

Ecrire les fonctions javascript suivantes dans le fichier fonctions.js :

  • Fonction 'chargement' ouvrant une boite de dialogue indiquant 'chargement de la page'
  • Fonction 'quitte' ouvrant une boite de dialogue indiquant 'page quittée'

Réactions

Faire en sorte que la fonction 'chargement' soit appelée au chargement de la page et que la fonction 'quitte' soit appelée lorsqu'on la quitte.

Une première fonction

Le but est d'écrire une page web permettant de vérifier si une donnée rentrée par un utilisateur est un nombre pair ou impair.

Préparation

Créer un fichier HTML vierge.

Créer dans ce fichier un formulaire contenant un champ de saisie pour du texte (rappel : input de type text...).

Le formulaire comprendra un bouton de soumission (rappel : input de type submit...).

Appel au javascript

Dans l'élément <form>, ajouter un attribut permettant de lancer la fonction parite() lors de la soumission du formulaire (voir les évènements).

Fonction parite

Définir une fonction javascript nommée parite().

Cette fonction doit récupérer la valeur entrée par l'utilisateur et afficher une boite de dialogue indiquant si le nombre entré est pair ou impair.

Récupération de la valeur :

Deux étapes :

  • Repérer l'élément de saisie (input text du formulaire) : pour cela on peut utiliser la méthode getElementById de l'objet getElementById> (document.getElementById("..."))
  • Récupérer la valeur à l'aide de l'attribut value. Voir la documentation du W3C sur le DOM appliqué aux éléments HTML : monelemment.value donne la valeur courante de monelement si monelement est un input

Une fois la valeur récupérée, tester si elle est paire ou impaire puis afficher le résultat dans une boite de dialogue
